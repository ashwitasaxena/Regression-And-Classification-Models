1. Regression Models 
1.1 Data for regression models - Boston Housing Data - can be found in library MASS
1.2 R Code for Regression Models is called Regression Models.R

2. Classification Models
2.1 Data for Classification models - German Credit Score Data - can be downloaded at the following link
http://archive.ics.uci.edu/ml/machine-learning-databases/statlog/german/german.data

3. The combined report with results from both regression models on Boston Housing data
and classification models on German Credit Score data can be found in 'Regression and Classification models report. docx'
